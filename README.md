# High Life Auto - "Cool to be Uncool"

Budget-friendly used car dealership website built with React + Vite.

## Features

- Live Frazer DMS inventory integration
- YouTube test drive video embedding
- Vehicle grading system
- "Honest Blemishes" transparency
- Mobile-first design

## Local Development

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```
